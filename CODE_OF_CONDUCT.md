# Code of Conduct

## Our Commitment

We are committed to a respectful, inclusive, and harassment-free environment for everyone participating in this project.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Provide technical feedback with clarity.
- Help newcomers with actionable guidance.

## Unacceptable Behavior

- Harassment, discrimination, or hate speech.
- Personal attacks, insults, or intimidation.
- Deliberate disruption or bad-faith conduct.
- Publishing private information without permission.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and pull requests that violate this Code of Conduct.

## Reporting

To report issues, contact the maintainer through GitHub issues/discussions and include relevant context.

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions, and other public communications connected to this repository.
