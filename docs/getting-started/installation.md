# Installation

## Requirements

- Node.js 18+
- npm 9+
- Discord bot token and application ID

## Install CLI

```bash
npm install -g arcane-builder
```

## Contributor Setup

```bash
git clone https://github.com/ramkrishna-js/arcane-builder.git
cd arcane-builder
npm install
npm link
```

## Verify

```bash
arcane --version
arcane --help
```
