# @arcane/leveling

XP and leveling progression systems.

## Typical Features

- XP on activity
- Rank progression
- Role rewards by level
- Leaderboards

## Example Config

```json
{
  "package": "@arcane/leveling",
  "version": "latest",
  "enabled": true,
  "config": {
    "xp": {
      "min": 10,
      "max": 25
    }
  }
}
```
