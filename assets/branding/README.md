# Arcane Branding Assets

## Files

- `assets/branding/logo.svg` - primary square logo
- `assets/branding/banner-horizontal.svg` - wide repository/documentation banner
- `assets/branding/banner-social.svg` - social card / OG preview banner

## Usage

Embed in markdown:

```md
![Arcane Builder](assets/branding/banner-horizontal.svg)
```

Use `logo.svg` for favicons/app icons after raster export.
